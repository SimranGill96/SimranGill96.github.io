Greenshot

Introduction

Greenshot is a screenshot software tool.It is avaliable for Windows and is easy to use for anyone.
It is a light-weight tool which helps in taking screenshots.
Greenshot can be used to create quick screenshots of a selected region,window or fullscreen.
One can easily add notes or highlight parts of the screenshots.
It is easy to export screenshots in various ways such as send to printer,
save to file or upload to photo sites like flickr,picasa and others.

General Usage

  1. Greenshot is used to create screenshots conveniently.
  2. It helps in editing screenshots according to one's usage.
  3. Screenshots can be either of a fullscreen or a selected area.
  4. Greenshot helps to use screenshots easily for printing, to email or upload on picture sites.
