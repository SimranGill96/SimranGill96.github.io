Mobile
The mobile format is most difficult to design because there are different screen sizes for different kinds of phones.
So one formatting is for smart phone while a 240x320px formatting has been added for small screen phones as some generation of phones have buttons which reduces the display area.
Tablets
Tablets are more of like small laptops or phones depending on their size,
while some tablets can have larger displays which adapt with rotation resulting in a desktop like display
so a base of 768px has been used in formatting so that screen is adjustable with rotation accordingly.
Desktops
The dimensions for the display of a desktop screen usually depends on the website that is being used either in fullscreen mode or minimised view.
A minimun width of 992px has been used so that if website is used in either of the view the layout is adjustable according to the screen.This makes it adaptable to the increase or decrease of the screen rather than leaving empty spaces on the screen.
